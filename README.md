# Food & Beverage Sales Dashboard

## Deskripsi
Project ini merupakan analisis data penjualan perusahaan distribusi 
makanan dan minuman menggunakan database Northwind. Tujuannya adalah 
menjawab pertanyaan bisnis terkait performa penjualan, kategori produk, 
dan kontribusi customer terhadap total revenue.

## Tools yang Digunakan
- SQL Server Management Studio — Query dan ekstraksi data
- Power BI — Visualisasi dan dashboard

## Pertanyaan Bisnis
1. Sales representative mana yang menghasilkan revenue tertinggi?
2. Kategori produk mana yang paling laris dan paling tinggi revenue-nya?
3. Siapa 10 customer teratas berdasarkan total belanja?

## Insight Utama
- Beverages dan Dairy Products konsisten menjadi kategori teratas 
  baik dari sisi units sold maupun revenue. Meat/Poultry memiliki revenue tinggi (peringkat 3) namun units sold rendah (peringkat 7), mengindikasikan harga produk yang tinggi
- Margaret Peacock adalah sales representatives dengan performa tertinggi (250K)
- Top 3 customer (Horst Kloss, Jose Pavarotti, Roland Mendel) 
  berkontribusi jauh di atas customer lainnya, mengindikasikan 
  ketergantungan tinggi pada segelintir customer

## Dataset
Northwind Database — Microsoft SQL Server Sample Database
